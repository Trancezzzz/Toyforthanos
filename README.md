# Toyforthanos

Online streaming providers for [Seanime](https://seanime.nyaa.dev).

## Providers

| Lang | ID | Name | Site | Status |
|------|-----|------|------|--------|
| EN | hianime | HiAnime | [hianime.ms](https://hianime.ms) | Active |
| IT | animeworld | AnimeWorld | [animeworld.ac](https://www.animeworld.ac) | Active |

## Install

In Seanime, go to **Settings → Online Streaming Providers** and add the manifest URL:

```
https://raw.githubusercontent.com/Trancezzzz/Toyforthanos/master/provider/en/hianime/manifest.json
https://raw.githubusercontent.com/Trancezzzz/Toyforthanos/master/provider/it/animeworld/manifest.json
```
